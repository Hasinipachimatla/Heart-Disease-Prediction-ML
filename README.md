# Heart Disease Prediction Using Machine Learning

## Project Overview

This project aims to predict the presence of heart disease using machine learning techniques. Three classification algorithms were trained and evaluated to determine the best-performing model.

## Problem Statement

Build, train, and evaluate machine learning models that predict heart disease based on patient health parameters. Compare multiple algorithms and identify the most effective model.

## Dataset

* **Dataset:** Heart Disease Dataset
* **Source:** https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Data Preprocessing

* Loaded and explored the dataset.
* Checked for missing values.
* Verified data types and categorical variables.
* Split the data into training and testing sets using an 80:20 ratio.

## Feature Engineering

* Performed correlation analysis using a heatmap.
* Analyzed feature importance using Random Forest.
* Retained all relevant features for model training.

## Machine Learning Models

1. Logistic Regression
2. K-Nearest Neighbors (KNN)
3. Random Forest

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

## Best Model Analysis

The three models were compared using classification metrics. The best-performing model was selected based on overall accuracy and F1 score. A confusion matrix was used to visualize prediction performance.

## Conclusion

* Three machine learning algorithms were successfully trained and evaluated.
* Performance was measured using Accuracy, Precision, Recall, and F1 Score.
* The best model achieved superior predictive performance.
* A confusion matrix was used to assess classification quality.
* The final model can assist in early detection of heart disease.
